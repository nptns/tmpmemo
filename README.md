This is a *ridiculously* simple Neovim plugin to make temporary notes

## Quick Start:
    git clone https://github.com/nptns/tmpmemo.git ~/.config/nvim/plugin/

## Keybindings:
    <leader> + m = New memo
    <ESC>        = Close memo
